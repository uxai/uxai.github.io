# Vajeh (واژه)

A simple web app for practicing Persian (Farsi) vocabulary by typing words using the standard Iranian keyboard layout.

The goal is to make learning Persian vocabulary fast, simple, and enjoyable through repeated typing practice.

## Features

- 🇮🇷 Practice common Persian vocabulary
- ⌨️ Type using the standard Persian keyboard layout
- 📈 Track typing speed (WPM)
- 🎯 Accuracy and mistake tracking
- 🌙 Clean, distraction-free interface

## Try it

Visit **https://uxai.github.io/** and start typing right away.

If you're running the project locally, it must be served from a local web server (rather than opening the HTML file directly). For example:

```bash
python -m http.server
```

Then open:

```
http://localhost:8000
```

## Vocabulary

The vocabulary is stored in a CSV file in this repository.
